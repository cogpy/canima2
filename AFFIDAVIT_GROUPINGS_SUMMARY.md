# Affidavit File Groupings - Summary

**Created:** November 2, 2025  
**Purpose:** Organized sections of Peter Faucitt's Founding Affidavit for easy access and conversion

## Location

All grouped files are located in: `/affidavit_grouped_files/`

## What Was Created

This implementation fulfills the request to create grouped sections of the founding affidavit for Peter Andrew Faucitt. The affidavit has been divided into 7 logical sections, each saved as a separate text file that can be easily converted to Word (.docx) or PDF format.

### Files Created

1. **INDEX.txt** - Complete table of contents mapping sections to paragraph markers
2. **README.md** - Instructions and overview of the directory
3. **Section_1-6_Legal_Opening.txt** (5.2 KB)
   - Introduction, Applicant, Identification of Respondents, Jurisdiction, Locus Standi, Relationships

4. **Section_7_Discovery_of_Financial_Irregularities.txt** (3.2 KB)
   - Discovery of irregularities, IT expenses, R500,000 payment

5. **Section_8_IT_Expenses_Analysis.txt** (1.6 KB)
   - Analysis of R6.7M (2024) and R2.1M (2025) IT expenses

6. **Section_9_Substantial_Financial_Misconduct.txt** (690 bytes)
   - Allegations of substantial financial misconduct

7. **Section_10_Delinquency_and_Probation.txt** (5.9 KB)
   - Legal grounds for delinquency/probation application

8. **Section_11_UK_Operations.txt** (1.3 KB)
   - UK operations and concerns about non-payment

9. **Section_12-18_Legal_Closing.txt** (8.4 KB)
   - Necessity of relief, interdicts, urgency, ex parte application, costs, prayer

## How to Use

Each text file can be:
1. Opened in any text editor
2. Copied and pasted into Microsoft Word or Google Docs
3. Saved as .docx format
4. Exported as PDF

Detailed instructions are provided in the INDEX.txt file within the directory.

## Benefits

- **Easy Navigation:** Find specific sections quickly using the index
- **Selective Sharing:** Share only relevant sections as needed
- **Format Flexibility:** Convert to Word or PDF as required
- **Preserved Content:** Exact wording from the original affidavit
- **Organized Structure:** Logical grouping of related content

## Reference

The content in these files corresponds to Peter Faucitt's Founding Affidavit filed on August 13, 2025, in Case 2025-137857 (High Court of South Africa, Gauteng Division, Pretoria).
