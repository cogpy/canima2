# Affidavit Grouped Files

This directory contains the founding affidavit of Peter Andrew Faucitt, divided into logical sections for easier review and conversion to Word/PDF documents.

## Contents

This directory contains 8 files:
- **INDEX.txt** - Table of contents mapping each section to its filename and paragraph markers
- **7 section files** - Each containing a specific portion of the affidavit

## File List

1. `Section_1-6_Legal_Opening.txt` - Introduction, Applicant, Identification of Respondents, Jurisdiction, Locus Standi, and Relationships
2. `Section_7_Discovery_of_Financial_Irregularities.txt` - Discovery of financial irregularities and the R500,000 payment
3. `Section_8_IT_Expenses_Analysis.txt` - Analysis of IT expenses (R6.7M in 2024, R2.1M in 2025)
4. `Section_9_Substantial_Financial_Misconduct.txt` - Allegations of substantial financial misconduct
5. `Section_10_Delinquency_and_Probation.txt` - Legal grounds for delinquency/probation application
6. `Section_11_UK_Operations.txt` - Discussion of UK operations and concerns
7. `Section_12-18_Legal_Closing.txt` - Necessity of relief, interdicts, urgency, ex parte application, costs, and prayer

## How to Use

To convert these text files to Word documents or PDFs:

1. Open the desired section file (e.g., `Section_7_Discovery_of_Financial_Irregularities.txt`)
2. Copy all the text
3. Open Microsoft Word or Google Docs
4. Paste the text into a blank document
5. Save as `.docx` format (File → Save As → Word Document)
6. For PDF: Save As → PDF (or File → Export → Create PDF/XPS)

## Purpose

These grouped files were created to:
- Provide easy access to specific sections of the affidavit
- Enable conversion to various document formats
- Facilitate review and analysis of different aspects of the case
- Allow for selective sharing of relevant sections
